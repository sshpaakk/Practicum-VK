# Practicum-VK
RuStore app android
# Инструкция к запуску приложения

1) Установить инструменты

      Android Studio 

      JDK 17+

      Android SDK 34+

2) Скачать проект на свой ПК

3) Открыть в Android Studio

      File → Open → выбрать папку проекта

      Android Studio автоматически скачает зависимости.

4) Запустить на эмуляторе

      Создать AVD (рекомендуется Android 14 / API 34)

      В верхней панели выбрать устройство

      Нажать Run
